This is a module for [IssabelPBX©](http://www.issabel.org/ "IssabelPBX Home Page"). [IssabelPBX](http://www.issabel.org/ "IssabelPBX Home Page") is an open source GUI (graphical user interface) that controls and manages [Asterisk©](http://www.asterisk.org/ "Asterisk Home Page") (PBX). IssabelPBX is licensed under GPL.
[IssabelPBX](http://www.issabel.org/ "IssabelPBX Home Page") is a completely modular GUI for Asterisk written in PHP and Javascript. Meaning you can easily write any module you can think of and distribute it free of cost to your clients so that they can take advantage of beneficial features in [Asterisk](http://www.asterisk.org/ "Asterisk Home Page")
